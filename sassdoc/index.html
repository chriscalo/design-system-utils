    sizing

    variables

    grid-unit

    $grid-unit: 8px !default;

    Description

    Size of base grid unit. All sizing is based on an 8px grid (overridable).

    Example

    Set padding to 1 grid unit

    .foo {
  padding: 1 * $grid-unit;  8px
}

    Used by

    • [function] gr-to-px
    • [mixin] stack-h
    • [mixin] stack-v
